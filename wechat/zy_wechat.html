# 资缘网微信端

资缘网微信端 —— 继电器项目微信端，包含买家端 + 微信云展。

项目主要运用 vue 相关技术栈 （Vue2 + vue-router + vuex），UI 框架采用

Vant (opens new window)（版本v2.8.1，请根据这个版本来下载依赖，新的版本部分组件会出现样式错乱）

# 其他

  1. 路由

    src/router

    meta 配置：

    • rightSiderBool 是否展示右侧悬浮（查一查，客服，TOP）
    • showCloudBottom 是否显示云展底部 tab
    • showBottom 是否显示首页底部 tab
    • showRelsySoBottom 是否显示 relayso 底部 tab
    • headerTitle 是否展示顶部导航 navBar
    • duibi 是否显示对照栏
    • check 是否显示查一查
    • requireAuth 是否需要登录权限

  2. 请求接口

    请求接口前要确保代理配置和本地 host 配置无误。

    • 代理配置：config/index.js

      module.exports = {
  dev: {

    // Paths
    assetsSubDirectory: 'static',
    assetsPublicPath: '/',
    proxyTable: {
      '/api': {
        target: 'http://gateway.zy.io:8700', // 接口的域名
        // secure: false,  // 如果是https接口，需要配置这个参数
        changeOrigin: true, // 如果接口跨域，需要进行这个参数配置
        pathRewrite: {
          '/api': '/api'
        }
      }
		...
    }
  }
}

    • 本地 host 配置：C:\Windows\System32\drivers\etc 下的 hosts 文件

      所有的 api 统一配置在 src/api文件下，根据具体的应用模块进行划分，如优惠券相关的接口统一写在 src/api/coupon.js 里面

      import { request } from '@/utils/request'
// 我的优惠券
export function currentUserCouponList(data) {
    return request({
        url: '/v1/order/c/orderMemberCouponReceive/currentUserCouponList',
        method: 'POST',
        data
    })
}

    注： 如需在本地连接测试环境进行调试，只需将 target设为https://m.relaysogo.com，并将 host 切换为测试环境。

  3. 适配方案

    使用 rem做页面适配，通过 postcss-pxtorem插件可以将 px 单位自动转换为 rem，具体在 src/.postcssrc.js中进行配置

    module.exports = {
  "plugins": {
    "postcss-import": {},
    "postcss-url": {},
    // to edit target browsers: use "browserslist" field in package.json
    "autoprefixer": {},
    "postcss-pxtorem": {
      // 设计稿 375:37.5
      // 设计稿：750:75
      // Vant 是基于 375
      rootValue: 75,
      propList: ["*"]
    }
  }
}js

    由于我们的设计稿都是 750px，所以将 rootValue 的值设为 75，在写样式时根据设计稿的 px 值来即可。

    需要注意的是 Vant 是基于 375px 的设计稿进行设计，所以在引用 Vant 的 ui 组件时需要在原来的像素大小上 *2

  4. css 预处理

    项目采用 less 作为编写样式的预处理器，在需要修改第三方样式出现不生效的情况，使用 /deep/进行样式穿透
