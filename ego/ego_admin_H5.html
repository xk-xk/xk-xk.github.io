# 商家端H5

此项目采用 uniapp (opens new window) 作为开发框架，开发过程中使用 HbuilderX 编辑器编写代码，运行到本地浏览器进行调试。

仓库地址：http://192.168.25.117/front/ego-merchant-mobile (opens new window)

# 运行

  1. 使用 HbuilderX 导入项目：点击工具栏 文件 => 导入 => 从本地目录导入
  2. 导入成功后点击工具栏 运行 => 运行到浏览器 => Chrome。

# 打包发布

  1. 点击 HbuilderX 工具栏 发行 => 网站-PC
  2. 网站标题设为一个空格（具体为什么我也不记得了），其他都不用操作，点击 发行
  3. 待控制台编译完成后会生成打包文件 /unpackage/dist/build/h5
  4. 登录 Xftp，开发环境：192.168.25.141，测试环境：192.168.25.145
  5. 进入 /opt/html 目录下，用第三步打包生成的 h5 文件夹进行替换

# 技术要点

  1. 项目打包发布前要确认 config/app-config.js 中配置正确：
