# 客户端打包

打包成客户端（桌面应用），总体分两步：第一步，打包可运行的安装包；第二步，将安装包打包成安装程序(Setup.exe)

注意：基于目前的打包方法打包出来的应用只有64位的操作系统能用，暂不支持32位操作系统！

# 第一步

  1. 从git仓库拉取打包安装包的项目：front-electron-standard (opens new window)，并安装依赖:

    npm install

  2. 配置打包应用：

    // /src/config/app.js

// 测试环境
exports.client = {
    identifier: "relaitest",
    name: "瑞莱E购（测试）",
    packageName: "com.relaitest.ego",
    icon: "./public/app_icon/relai/app.ico"
}

// 瑞莱生产环境
// exports.client = {
//     identifier: "relai",
//     name: "瑞莱电子E购",
//     packageName: "com.relai.ego",
//     icon: "./public/app_icon/relai/app.ico"
// }

// 资缘生产环境
// exports.client = {
//     identifier: "zy",
//     name: "A生产企业E购",
//     packageName: "com.zy.ego",
//     icon: "./public/app_icon/relai/app.ico"
// }

// 需要打包对应的租户商家，配置对应的属性即可，比如需要打包汇港的就安装以上的示范配置成如下：
// exports.client = {
//     identifier: "hke",
//     name: "汇港",
//     packageName: "com.hke.ego",
//     icon: "./public/app_icon/hke/app.ico"
// }运行：

  3. 运行：

    npm run start

    运行成功后会自动弹出客户端应用界面，退出终端即关闭正在运行的客户端

  4. 打包：

    npm run make

    在 /out文件目录下生成对应的安装包后说明打包成功，此时终端仍在运行不必理会，可以直接终止。

    注意输出的文件名是中文的要改成英文，如relaitest-win32-x64，否在在执行第二步的时候可能会报错。

在执行第二步之前需要下载第三方软件NSIS，参考文章：https://www.cnblogs.com/zp9628/p/11634756.html (opens new window)

步骤如下：

  1. 打开VNISEdit 编译环境 软件，然后打开“文件—新建脚本：（向导）”，点击下一步

  2. 填写应用信息，这里我们只需要填写应用名称和版本号，其他放空，点击下一步

  3. 直接点击下一步

  4. 点击下一步

  5. 不需要授权文件，删除，点击下一步

  6. 选择应用程序文件，这里需要配置两个，一个是到应用程序如：D:\workspace\front-electron-standard\out\relaitest-win32-x64\瑞莱E购（测试）.exe，

    一个是到应用程序所在的文件夹，如：D:\workspace\front-electron-standard\out\relaitest-win32-x64\ ，注意这里是选取文件夹下的所有文件，需要在后面加上 *.*

  7. 下一步

  8. 下一步

  9. 下一步

  10. 点击完成，选择

  11. 执行成功后会在自动执行安装程序，按照步骤一步步安装即可，并且同时会在指定的文件夹生成一个打包源码xxx.nis和安装程序Setup.exe，运行打包源码程序后可以再次在编辑器中重新打包安装程序。
